Scan completed surveys
  1. Scan the surveys using the following scanner settings:
    • monochromatic
    • 300 dpi
    • Page Size: letter
    • Fit to page
    • PDF Format
  2. Zip all the scanned PDF surveys into a .zip file (WARNING: Make sure the .zip file contains the PDF files, and NOT a folder containing the files)
Upload the zip
  1. Click the “Trainings” tab in the navigation bar
  2. Click on the name of the training for which you want to upload surveys
  3. Click on "Upload Complete Surveys" in the action panel
  4. Use the file chooser to select the .zip file that contains the completed survey PDFs and select it
  5. Cick "Upload"
Upload the results
  1. Click the “Trainings” tab in the navigation bar
  2. Click on the name of the training for which you want to upload surveys
  3. Take note of the "Training ID" field displayed in the information panel
  4. Use this link to go to the QueXF admin panel
  5. Find the row where the "Questionnaire" field matches the Training ID for this training
  6. Click on the "CSV" link (NOT "CSV Labelled") and download and save the .csv file that is produced
  7. Close the QueXF admin panel tab and return to the training for which you want to upload surveys
  8. Click on "Upload Results CSV" in the action panel
  9. Use the file chooser to select the .csv file that you downloaded
  10. Fill in the "Attendance" field with the official attendance for this training
  11. Click "Analyze"
