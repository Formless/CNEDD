Adding data
  1. Export the membership data from your database for the month to a .csv file
  2. Click the "Membership Data" tab in the navigation bar
  3. Click on "New Membership Report" in the actions panel
  4. Use the file chooser to select the membership .csv file
  5. Enter the month and year that this data is for
  6. Click "Upload"
  7. You can now view, edit, or analyze the data
